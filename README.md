# Controle de Gastos

---

# Tecnologias utilizadas

## Backend

* **.NET / ASP.NET Web API**
* **Entity Framework Core**
* **SQLite**

## Frontend

* **React**
* **TypeScript**
* **Vite**
* **Axios**
* **TailwindCSS**

---

# Como executar o projeto

## Clonar o repositório

```
git clone https://github.com/JoaoCamo/controle-gastos.git
cd controle-gastos
```

---

# Backend

Entre na pasta do backend:

```
cd backend
```

Restaure as dependências:

```
dotnet restore
```

Execute a API:

```
dotnet run
```

A API será iniciada em:

```
https://localhost:5267
```

O banco de dados SQLite é criado automaticamente através das migrations.

---

# Frontend

Entre na pasta do frontend:

```
cd frontend
```

Instale as dependências:

```
npm install
```

Execute o projeto:

```
npm run dev
```

O frontend será iniciado em:

```
http://localhost:5173
```
